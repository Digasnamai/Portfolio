Modular City

About

This project was created on the theme of "transmediation". This text-based adventure game is based on Ray Bradbury's story "The Veldt," which is a cautionary tale about a futuristic family living in a fully automated house that caters to their every need.
The user follows the events of the story from the perspective of the house via a retro computer terminal, while the computer sends the user queries to which the user must respond in order to proceed. In this project, I was in charge of constructing the website and all the interactions, rewriting the story, audio editing and implementation, and putting it live on the web.

Project Info

Roles: Development, texturing

Team Size: 1 people

Time frame: 3 weeks
